# Nodejs-ci-cd-starter
Student REST API & CI/CD Pipeline | Node.js, Express.js, Jest, Supertest, GitHub Actions

Built a RESTful backend API with modular CRUD endpoints and standardized JSON response structures.
Implemented automated integration testing using Jest and Supertest to ensure API reliability.
Designed a Continuous Integration (CI/CD) pipeline using GitHub Actions to automatically run the test suite on every repository push.
